☁️ Sincronização automática entre dispositivos

Seus dados ficam num Gist privado no GitHub. Ao abrir o app em qualquer dispositivo configurado, os dados são atualizados sozinhos. Criar token aqui (marca só "gist", expiração: "No expiration").

🤖 Como funciona: ao abrir → baixa atualizações novas. Ao alterar → sobe em segundo plano (1.5s depois). A cada 1 minuto → verifica novamente. Quando dois dispositivos editam ao mesmo tempo, aparece um alerta pra você escolher qual versão manter.
